## Py-TicTacToe-with-db
TicTacToe PvP game in Python with database support

## Prerequisite
pretty table - pip3 install prettytable

## How to use
* Clone/download this repository http://github.com/sunx2/Py-TicTacToe-with-db 
* Unzip the file
* Run the main.py
